Tags HTML
Tag Como utilizar
header Cabeçalho: Representa um grupo de introdução à página, podendo conter elementos como a logo, formulário de pesquisa, e outros.
nav Elemento de navegação, uma seção que aponta para outras páginas ou áreas dentro da mesma página, com links de navegação.
ul Unordered list: Lista de elementos desordenados, ou seja, onde a ordenação numérica não importa.
ol Ordered list: Lista de elementos ordenados, onde a ordenação numérica importa.
li List item: Item de uma lista, seja ordenada ou desordenada. Deve estar contido em um elemento pai do tipo ul, ol, ou menu.
a Elemento Âncora: Cria uma hiperligação nas páginas web, para arquivos, endereços de email, telefone ou endereços URL.
aside Representa uma seção que é tangencialmente relacionada com o conteúdo, normalmente representada como barras laterais.
h1 - h6 Os elementos de h1 até h6 representam os seis níveis de títulos de uma seção, h1 é o nível mais alto, e h6 é o mais baixo.
abbr Elemento de abreviação html representa a abreviação e opcionalmente fornece a descrição completa para ela. O atributo "title", deve conter a descrição completa e apenas ela.
img Representa a inserção de uma imagem no documento.
main Define o conteúdo principal dentro do body no documento ou aplicação.
section Representa uma seção genérica contida no documento, geralmente são identificados através de um título de cabeçalho.
article Representa uma composição independente em um documento, destinado a ser distribuído de forma independente.
div Representa um elemento de divisão, ou seja, um container genérico que não representa nada, não possuindo valor semântico, de significado. Utilizado para agrupar elementos para fins de estilização. Deve ser utilizado quando não houver outro elemento de semântica.
footer Rodapé: normalmente contém informações sobre o autor da seção de dados, direitos autorais ou links para documentos relacionados.
address Fornece informações de contato e pode incluir qualquer tipo de informação de contato necessária, como endereço físico, URL, email, telefone, mídia social, coordenadas geográficas, e outros.
